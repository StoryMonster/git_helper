gitHelper v1.0

dochen 2018-3-30

gitHelper is a convenient tool to help you work with git, it will help you quickly get the result you want and apply your modification

Guide

git log | githelper find [keyword] ### this will help you quickly find the entire commit information you want, and the commit id ahead or behind it

git pull --rebase | githelper readconflict ### this will help collect all conflicts in to a file, in that file you can solve all conflicts, and then apply your solution to your project

githelper applyconflict ### apply the modifications you modified into your project
